# ToDoList

![C++](https://img.shields.io/badge/language-C++-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

> A lightweight, console-based task manager in C++

Allows you to add, delete, and view tasks with deadlines. All task data is stored locally in JSON format.

- Colorful alerts and notifications  
- Smooth text animations  
- Local JSON storage for all tasks  

---

## Features

- **Add Task** – Enter the task name and end date.  
- **Delete Task** – Remove a task by its name.  
- **Show Tasks** – Display all tasks with creation and end dates.  
- **Alerts & Notifications** – Receive feedback about successful actions or errors.  
- **Console-friendly Interface** – Easy-to-read, colorful outputs.  

## Custom Library

This project contains a small custom library [console_writer](https://github.com/dixe1/UsefulFunctions/tree/main/console_writer) for printing text with optional color and delays.
